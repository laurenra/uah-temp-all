UAH Global Satellite Temperature Time Series

Satellite data file from Global Temperature Report of the University of Alabama Huntsville is updated monthly.

Download Data

You can link to this page with query parameters separated by "&" to show or hide specific plot lines.

For example, to hide the (default) Global Average (ga) plot and show the Tropics All (ta) and Northern Hemisphere All (na) plots:

http://yburbs.com/uah-temp-all/uah-temp-all.html?ga=0&ta&na

or

http://yburbs.com/uah-temp-all/uah-temp-all.html?ga=0&ta=1&na=1
